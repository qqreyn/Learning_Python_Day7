# Learning_Python_Day7

Today's topic is building a complete project!

**Goal:** Build a complete console-based task manager using all Week 1 concepts

**Project:**
Console Task Manager:  
Create a full program with a menu:

1. Add task  
2. View all tasks  
3. Mark task done (remove by number)  
4. Count tasks  
5. Clear all  
6. Exit  

- Store tasks in a list  
- Show task numbers when viewing tasks  
- Validate user input  
- Keep looping until the user chooses Exit  
- Make the program fully functional using everything learned this week
